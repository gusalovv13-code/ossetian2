# Исправление рекламы — v1.11.4

## Что исправлено

- Реклама в ленте теперь появляется и при небольшом количестве товаров. Раньше при интервале 6 и пяти товарах рекламный блок не вставлялся вообще.
- Дата и время начала/окончания кампании сохраняются с правильным часовым поясом пользователя.
- После создания или изменения кампании клиент и прокси не используют старый ответ `/api/ads`.
- В административной панели под каждой кампанией выводится причина текущего состояния показа.

## Как быстро проверить

1. Откройте **Администратор → Реклама**.
2. Создайте кампанию и установите статус **Активна**.
3. Для немедленного запуска оставьте поля «Начало» и «Окончание» пустыми.
4. Укажите либо внешнюю ссылку, начинающуюся с `https://`, либо ID существующего объявления.
5. Выберите размещение:
   - `Верх каталога` — баннер над категориями;
   - `В ленте товаров` — вставка между товарами;
   - `В карточке товара` — блок внутри открытого объявления.
6. Сохраните кампанию и заново откройте каталог.

URL изображения необязателен. Без него показывается стандартная рекламная заглушка.
