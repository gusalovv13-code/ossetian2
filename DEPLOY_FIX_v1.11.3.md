# v1.11.3 — ускорение деплоя Render

- Исправлены четыре недоступные внешнему серверу ссылки в `package-lock.json`.
- Все зависимости теперь загружаются с `https://registry.npmjs.org/`.
- Удалены чрезмерные `fetch-retries` и тайм-аут 600000 мс, которые маскировали ошибку и задерживали сборку.
- Отключены `npm audit` и `npm fund` во время production-сборки.
- Исключены тесты и Markdown-документация из Docker build context.
