# Алания Маркет v1.11.2 — устранение зависаний каталога

## Устранённая причина

В версии 1.11.1 старые объявления без отдельной миниатюры всё ещё попадали в публичный JSON с исходной Base64-фотографией. Из-за этого ответ каталога мог весить десятки мегабайт. Telegram WebView синхронно разбирал большой JSON и декодировал изображения, поэтому задерживались даже обычные нажатия.

## Изменения

1. Публичные JSON-ответы больше не содержат Base64-фотографии.
2. Миниатюры выдаются через `/api/products/:id/thumbnail`.
3. Полные изображения выдаются через `/api/products/:id/media/:index`.
4. Изображения получают длительный HTTP-кеш и версионируются временем обновления объявления.
5. Карточка товара возвращает текст и характеристики отдельно от изображений.
6. Каталог больше не загружается на главной странице в фоне.
7. Запрос списка выполняется одним SQL-запросом без отдельного `COUNT(*)`.
8. Добавлены тайм-ауты PostgreSQL, понятная ошибка загрузки и кнопка повтора.
9. Для интерактивных элементов включён `touch-action: manipulation`.
