# Осетинский маркет v1.12.1

## Что исправлено

1. **Каталог снова загружает объявления.**
   В v1.12.0 запрос без текста поиска формировал в PostgreSQL `ORDER BY 0`, из-за чего `/api/products` отвечал ошибкой 500. Сортировка теперь собирается без фиктивного нулевого выражения.

2. **Выбор категории восстановлен.**
   Кнопки категорий больше не сжимаются до ширины эмодзи, нормально прокручиваются по горизонтали и обрабатываются одним устойчивым обработчиком событий.

3. **Раздел «Мои объявления» переработан.**
   Карточки показываются по одной в строке. Кнопки управления расположены сеткой 2×2 и имеют подписи:
   - Редактировать;
   - Продано / Опубликовать;
   - Выделить цветом / Заявка отправлена;
   - Удалить.

4. **Исправлено отображение вкладок.**
   «Активные», «Проданные», «Черновики» и «Архив» помещаются на мобильном экране; на очень узких устройствах автоматически переходят в сетку 2×2.

5. **Обновлён кеш Telegram WebView.**
   JS и CSS подключаются с версией `1.12.1`, чтобы Telegram не оставлял старый интерфейс после развёртывания.

## Проверка

- `npm run check` — успешно;
- `npm test` — 73 из 73 тестов успешно;
- ZIP проверен после упаковки.
