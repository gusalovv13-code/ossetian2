# Осетинский маркет v1.12.5

## Что изменено

- В платном выделении оставлены три цвета: фиолетовый, зелёный и золотой.
- Фиолетовый установлен как основной цвет по умолчанию и согласован с дизайном приложения.
- Перед отправкой заявки пользователь выбирает цвет в отдельном аккуратном окне.
- Реклама в каталоге снова занимает одну ячейку и по размеру соответствует обычному объявлению.
- В разделе администратора «Выделение» выбранный цвет показан отдельным цветным индикатором.
- В каждой заявке добавлена кнопка «Написать в Telegram». Она открывает чат по username, а при его отсутствии — профиль по Telegram ID и копирует подготовленный текст об оплате.
- Во всём приложении установлен единый системный шрифт без тяжёлых и визуально резких начертаний.
- Старые синие или некорректные варианты выделения автоматически переводятся в фиолетовый.
- Выделенные объявления продолжают сортироваться по обычной дате и не поднимаются выше остальных.

## Проверка версии

После развёртывания `/api/version` должен вернуть:

```json
{
  "ok": true,
  "version": "1.12.5",
  "catalogOrderFix": true,
  "build": "feature-color-chat-fonts"
}
```
