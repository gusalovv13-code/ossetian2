# Алания Маркет v1.12.7

## Что изменено

1. Username Telegram-бота установлен: `os_15market_bot`.
2. При нажатии «Поделиться» приложение запрашивает у сервера подготовленное Telegram-сообщение и открывает нативный выбор чата.
3. В чат отправляются:
   - фотография объявления;
   - название и цена;
   - местоположение, если оно заполнено;
   - прямая ссылка на объявление;
   - кнопка «Открыть объявление».
4. Исходные WebP, PNG и JPEG преобразуются сервером в JPEG до 1280 px и не превышают лимит Telegram.
5. Для старых клиентов используется страница `/share/product/:id` с Open Graph-фотографией и автоматическим возвратом в Telegram.
6. Файл `.env` уже содержит `BOT_USERNAME=os_15market_bot`. Секретные значения в него не добавлялись.

## Развёртывание

После замены файлов выполните:

```bash
npm ci
npm start
```

При использовании нестандартного reverse proxy можно задать:

```env
PUBLIC_BASE_URL=https://ваш-домен.example
```

Миграция PostgreSQL для этой версии не требуется.
