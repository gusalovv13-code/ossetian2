# Алания Маркет v1.13.0

## Новое

- Поиск получил зависимые списки района, типа товара, марки, модели и года выпуска.
- Для категории «Авто» продавец выбирает тип транспорта, марку, модель и год.
- Для категории «Электроника» продавец выбирает тип устройства, бренд, модель и год.
- Если значения нет в справочнике, можно выбрать «Другое» и ввести его вручную.
- Структурированные значения сохраняются в характеристиках объявления и участвуют в точной серверной фильтрации.

Миграция базы данных не требуется.
