# Алания Маркет v1.13.4

## Плавная навигация

- Добавлены короткие аппаратно-ускоренные переходы между экранами приложения.
- При открытии следующего экрана используется мягкое движение справа, при возврате — слева.
- Заголовок и активный пункт нижней навигации переключаются плавно.
- Диалоги и полноэкранный просмотр фотографий открываются без резкого появления.
- Повторные быстрые нажатия безопасно очищают предыдущую анимацию и не блокируют интерфейс.
- Учитывается системная настройка `prefers-reduced-motion`.

Изменения базы данных не требуются.
