# Алания Маркет v1.13.5

## Telegram Mini App

- Возвращён автоматический запрос полноэкранного режима при открытии приложения.
- В полноэкранном режиме снова используется компактная прозрачная верхняя панель Telegram.
- При возвращении в приложение fullscreen запрашивается повторно.
- Вертикальное сворачивание остаётся заблокированным.
- Для старых клиентов Telegram без поддержки fullscreen сохранён обычный расширенный режим и корректное положение заголовка.

## Интерфейс

- Плавные переходы между страницами из v1.13.4 сохранены.
- Изменений базы данных нет.
