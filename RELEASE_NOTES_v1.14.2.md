# Алания Маркет v1.14.2

- Карточки в блоках «Другие объявления продавца» и «Похожие объявления» стали компактнее: теперь на экране помещается почти две карточки, уменьшены фото, отступы и служебные плашки.
- При возврате из карточки объявления каталог восстанавливает прежнюю позицию прокрутки.
- Кнопка «Позвонить» теперь является настоящей ссылкой `tel:`; для старого закэшированного HTML добавлен безопасный fallback.
