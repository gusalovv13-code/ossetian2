# Алания Маркет v1.17.0

## Умный поиск

Поиск теперь расширяет каждый поисковый термин синонимами и транслитерацией. Для PostgreSQL, где доступно расширение `pg_trgm`, дополнительно включается поиск по небольшим опечаткам через `word_similarity`. Если расширение недоступно, сервер продолжает работать с обычным поиском, синонимами и транслитерацией.

В базовый словарь включены распространённые варианты для электроники, авто, вакансий и локальных названий Владикавказ / Дзауджикау / Дзæуджыхъæу.

## Профессиональные продавцы

Пользователь может включить бизнес-профиль и указать:

- название магазина или компании;
- сферу деятельности;
- адрес;
- график работы;
- сайт.

Бизнес-профиль получает отдельный бейдж и витрину. Если администратор уже выдал продавцу повышенный лимит объявлений, такой бизнес автоматически отображается как проверенный.

## Безопасность

Добавлены:

- Content-Security-Policy с разрешением Telegram WebApp;
- HSTS для HTTPS-развёртываний;
- X-Content-Type-Options, Referrer-Policy и Permissions-Policy;
- базовый in-memory rate limiting для API, мутаций и поиска;
- ограничение запроса по `Content-Length` до 32 МБ до JSON-парсинга;
- настраиваемые лимиты через переменные окружения.

Для горизонтально масштабируемого развёртывания с несколькими экземплярами приложения in-memory rate limiting следует в будущем заменить на общий Redis/PostgreSQL-backed limiter.
