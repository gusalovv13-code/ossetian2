# Алания Маркет v1.18.0

## Что добавлено

- Расширенная система доверия продавца: рейтинг, возраст профиля, подтверждения, продажи, просмотры, избранное и активные жалобы.
- Платное продвижение с тарифами `boost`, `vip`, `premium`, сроком, ценой и приоритетом выдачи. Оплата пока оформляется как заявка и подтверждается администратором; платёжный шлюз не подключён.
- AI-модерация поверх локальных правил. При отсутствии AI-ключа публикация продолжает работать на локальной модерации.
- AI-помощник создания объявления по первой фотографии с безопасным локальным fallback.
- Расширенная витрина профессионального продавца.
- Аналитика продавца: просмотры, избранное, активные/проданные объявления, продвижения, динамика уникальных просмотров и лучшие товары.
- Дополнительная защита API: проверка Origin для изменяющих запросов, Request ID, отдельный rate limit для AI, журнал security events с хешированными сетевыми идентификаторами.

## Настройка AI

Добавьте в окружение:

```env
OPENAI_API_KEY=...
OPENAI_MODEL=gpt-5.6
AI_LISTING_ASSISTANT_ENABLED=true
AI_MODERATION_ENABLED=true
AI_TIMEOUT_MS=20000
AI_RATE_LIMIT_MAX=20
```

Без `OPENAI_API_KEY` приложение остаётся работоспособным: AI-модерация пропускается, а помощник объявления возвращает локальную подсказку.

## Продвижение

Цены и сроки можно менять через `.env`:

```env
PROMO_BOOST_PRICE_RUB=99
PROMO_BOOST_DAYS=1
PROMO_VIP_PRICE_RUB=299
PROMO_VIP_DAYS=7
PROMO_PREMIUM_PRICE_RUB=599
PROMO_PREMIUM_DAYS=14
```
