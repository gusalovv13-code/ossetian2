# Алания Маркет v1.19.0 — Production Readiness

## Что добавлено

- Онлайн-оплата продвижения через YooKassa с автоматической активацией после подтверждённого платежа.
- История платежей и fallback на ручную заявку, если платёжный провайдер не настроен.
- Верификация профессиональных продавцов с очередью в админке и статусом «Проверенный бизнес».
- AI-модерация 2.0: настраиваемые пороги риска, отображение AI-оценки, учёт дневного бюджета и раздельные модели для публикаций/модерации.
- Синхронизация базового пакета правил модерации РФ.
- Расширенная аналитика продавца: звонки, сообщения, репосты, конверсия и динамика просмотров.
- Второй фактор доступа к админским API через SHA-256 секретного кода.
- Логические gzip-backup PostgreSQL с ротацией; ручной запуск из админки и CLI `npm run backup`.
- Раздельные юридические согласия для ПД, пользовательского соглашения и публичных контактов.
- Мобильная стабилизация интерфейса админки и новых диалогов.

## Настройка production

Для онлайн-оплаты заполните `YOOKASSA_SHOP_ID`, `YOOKASSA_SECRET_KEY` и корректный HTTPS `PUBLIC_BASE_URL`.
Для AI задайте `OPENAI_API_KEY`; без него локальный fallback продолжает работать.
Для автоматических резервных копий включите `AUTO_BACKUP_ENABLED=true`.
Для дополнительной защиты админки задайте `ADMIN_ACCESS_CODE_SHA256`.
