# Статус проверки v1.15.0

## Успешно

- `npm run check` — синтаксис сервера, Telegram-авторизации, модулей модерации и клиентского скрипта корректен.
- `npm run test:moderation` — 4 из 4 новых тестов модерации проходят.
- Проверено 99 уникальных системных правил без повторяющихся ID и комбинаций «выражение + тип».

## Старый общий набор тестов

Полный `npm test` содержит исторические проверки нескольких прошлых версий, которые одновременно требуют несовместимые номера кеша и старое поведение интерфейса. В исходном архиве до изменений уже было 44 падающих теста. В текущем рабочем варианте — 50; дополнительные 6 относятся к ранее изменённым звонкам и скидкам, а не к модулю модерации.

Для проверки этой версии модерации используйте `npm run test:moderation` и `npm run check`.
