OSSETIAN MARKET v1.14.1 — КАК ЗАГРУЗИТЬ

1. Распакуйте архив.
2. Загрузите содержимое папки в корень ветки main репозитория GitHub.
3. Не загружайте сам ZIP как единственный файл.
4. На Render выполните Manual Deploy -> Clear build cache & deploy.
5. В новом логе должны появиться строки:
   ossetian-market@1.14.1 start
   Server started on 0.0.0.0:...

Важно: версия 1.14.1 не завершает процесс при временно недоступной PostgreSQL,
но база данных всё равно должна иметь статус Available, а DATABASE_URL должен
содержать её текущий адрес подключения.
